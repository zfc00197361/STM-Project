1 Usage of the STM library on Linux.<br>
2 In order to use the O_STM library with any C++ application, it need to be placed to the operation system /usr/lib directory.<br>
3 1. Copy lib_o_stm.so file to /usr/lib: sudo cp lib_o_stm.so /usr/lib
4 2. Include the TM.h TX.h and the OSTM.h files in your application.<br>
5 3. Create Makefile :
6 <br><br>
7 ### Makefile.mk Documentation<br><br>
8 EXE =Test<br>
9 CC = g++<br>
10 PROGRAM = app<br>
11 CFLAGS =-std=c++14 -pthread <br>
12 CFILES = main.cpp AIB.cpp ULSTER.cpp BOA.cpp UNBL.cpp SWBPLC.cpp<br>
13 HFILES = TM.h TX.h OSTM.h AIB.h ULSTER.h BOA.h UNBL.h SWBPLC.h<br>
14 <br><br>
15 all:$(PROGRAM)<br>
16 <br><br>
17 ###Rule for SHARED linking<br>
18 $(PROGRAM):$(CFILES) $(HFILES)<br>
19  $(CC) $(CFLAGS) *.cpp -I -L /usr/lib/lib_o_stm.so -o $(EXE) <br>
20 clean:<br>
21  rm -f *.o<br>
22 <br>
23 3. Run the application/executable file : ./Test
